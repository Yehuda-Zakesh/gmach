# תיקיית התוכנות / software folder

כאן מניחים את קבצי התוכנה שיוצגו בספרייה.

## איך זה עובד

1. העתיקו לכאן קבצים (EXE, ZIP, PDF ועוד) או תיקיות שלמות.
2. פתחו את `admin.html` ולחצו על **סריקת תיקייה**.
3. הפריטים החדשים יתווספו אוטומטית — ערכו אותם ולחצו **שמירה**.

## כללים חשובים

- **עומק תיקיות** — נתמך ללא הגבלה מעשית (עד 12 רמות).
- **שמות בעברית** — נתמכים במלואם.
- **אין למחוק קבצים** מבלי לנקות אותם גם בממשק הניהול; אחרת הם יסומנו כ"חסרים".
- קבצים שמתחילים בנקודה (`.gitignore`), וכן `Thumbs.db` ו־`desktop.ini`, מדולגים בסריקה.

## התנהגות תיקיות

לכל תיקייה אפשר להגדיר בממשק הניהול אחת משתי התנהגויות:

| מצב | מה המשתמש רואה |
| --- | --- |
| **קטגוריה** | לחיצה על התיקייה פותחת את תוכנה, והקבצים שבתוכה מתנהגים כמו כל קובץ אחר. |
| **חבילה** | התיקייה מוצגת ככרטיס אחד עם כפתור הורדה יחיד. |

הקובץ הזה עצמו מדולג בסריקה ואינו מוצג בספרייה.
