# תיקיית התמונות / images folder

תמונות שמועלות דרך ממשק הניהול נשמרות כאן אוטומטית:

```
images/
├── icons/        אייקונים של תוכנות (מוקטנים ל־256px)
└── thumbnails/   תמונות ראשיות לכרטיסים (מוקטנות ל־800px)
```

## הערות

- **אין צורך לנהל את התיקייה הזו ידנית.** ממשק הניהול יוצר את תתי־התיקיות ואת שמות
  הקבצים בעצמו.
- כל תמונה נשמרת בנתיב **יחסי** בלבד, כך שהיא ממשיכה לעבוד בכל אות כונן.
- בדפדפנים שאינם תומכים בכתיבה ישירה לקבצים (כגון Firefox), התמונה מוטמעת
  ישירות בתוך מסד הנתונים כ־data URL במקום להישמר כאן. זה עובד באותה מידה,
  אך מגדיל את קובץ הנתונים.
- מחיקת תמונה מכאן תגרום לכך שהכרטיס יציג את אייקון ברירת המחדל של סוג הקובץ.
  זה לא ישבור את הספרייה.
